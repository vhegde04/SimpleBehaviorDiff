BehaviorDiff emerged from work on MirrorBench, a benchmark for measuring model self-consistency, and feeds into CORE (Contextual Observation and Response Engine), a semantic state management layer for trajectory-aware AI safety. The behavioral fingerprinting techniques developed here directly inform Askli's posture engine.
